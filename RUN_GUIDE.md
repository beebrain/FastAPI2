## 🚀 วิธีการรันระบบ (How to Run)

โปรเจกต์นี้มี 2 โหมดหลัก คุณสามารถเลือกโหมดที่ต้องการรันได้ดังนี้:

### 1. โหมดมาตรฐาน (General Object Detection)
ใช้โมเดล `yolov8n.pt` สำหรับตรวจจับวัตถุทั่วไป 80 ชนิด (คน, รถ, สัตว์ ฯลฯ)
- **คำสั่งรัน**:
  ```bash
  python main.py
  ```
- **การเข้าใช้งาน**: [http://localhost:8000](http://localhost:8000)

### 2. โหมดตรวจจับกากบาท (Cross Detection)
ใช้โมเดล `best_cross.pt` ที่ผ่านการเทรนมาเฉพาะทางเพื่อตรวจจับสัญลักษณ์กากบาท
- **คำสั่งรัน**:
  ```bash
  python main_cross.py
  ```
- **การเข้าใช้งาน**: [http://localhost:8001](http://localhost:8001)

## 🖥️ การใช้งานหน้าเว็บ (Web Interface)

1. **Upload**: ลากไฟล์รูปภาพมาวางหรือคลิกที่พื้นที่ **"อัพโหลดรูปภาพ"**
2. **Confidence Threshold**: ปรับค่าความเชื่อมั่น (0.05 - 1.0)
   - *ค่าน้อย*: ตรวจจับได้เยอะ แต่อาจมีผิดพลาด (False Positive)
   - *ค่ามาก*: แม่นยำสูง แต่อาจจะหาไม่เจอหากภาพไม่ชัด
3. **Detect**: กดปุ่ม **"🚀 เริ่ม Detect"** เพื่อส่งภาพไปประมวลผล
4. **Results**:
   - ระบบจะแสดงภาพเปรียบเทียบระหว่าง **"ต้นฉบับ"** และ **"ผลการ Detect"**
   - มีตารางสรุปจำนวนวัตถุและค่าพิกัด (Bounding Box)
   - สามารถกดปุ่ม **"💾 ดาวน์โหลด"** เพื่อบันทึกภาพผลลัพธ์ได้

## 🆘 การแก้ไขปัญหาเบื้องต้น (Troubleshooting)

- **พอร์ตชนกัน (Port Conflict)**: หากรันไม่ได้เพราะพอร์ตถูกใช้ไปแล้ว ให้ปิดเซิร์ฟเวอร์เก่าด้วย `Ctrl + C` หรือเปลี่ยนเลขพอร์ตท้ายไฟล์ `.py`
- **โมเดลหาย**: ตรวจสอบว่าไฟล์ `yolov8n.pt` หรือ `best_cross.pt` อยู่ในโฟลเดอร์เดียวกับสคริปต์
- **ModuleNotFoundError**: ให้รัน `pip install -r requirements.txt` อีกครั้งเพื่อให้แน่ใจว่าติดตั้ง library ครบแล้ว
